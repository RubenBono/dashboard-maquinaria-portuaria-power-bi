# Dashboard Análisis Maquinaria Portuaria

## Descripción

Dashboard desarrollado en Microsoft Power BI para el análisis de aciertos y errores de maquinaria del puerto.

## Tecnologías utilizadas

- Microsoft Power BI.
- Microsoft Excel.
- Entorno: Sector portuario / logístico.

## Funcionalidades

- Visualización de KPIs de rendimiento de maquinaria.
- Análisis de errores e incidencias por equipo.
- Seguimiento de aciertos operativos en tiempo real.

## Contexto

Proyecto desarrollado en entorno dentro del sector portuario. Los datos han sido anonimizados.
